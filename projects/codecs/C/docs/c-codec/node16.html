Environment initialization and cleanup

Most environments need to store an internal state representation and therefore many environment programs you write will need to allocate and deallocate data structures before and after a learning experiment. The env_init function allocates any global data structures and variables that will be accessed by the start and step functions. For example, the env_init method might initialize the tabular state variable to zero and allocate a numStates X numStates state transition array. The env_cleanup function usually deallocates or frees anything allocated in env_init.
