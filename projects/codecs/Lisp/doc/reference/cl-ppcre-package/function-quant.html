package cl-ppcre
( quant < lexer > )
Parses and consumes a <quant>.
The productions are: <quant> -> <greedy-quant> | <greedy-quant>“?”.
Will return the <parse-tree> returned by GREEDY-QUANT and optionally
change :GREEDY-REPETITION to :NON-GREEDY-REPETITION.
