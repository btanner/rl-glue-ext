package cl-ppcre
( reg-expr < lexer > )
Parses and consumes a <regex>, a complete regular expression.
The productions are: <regex> -> <seq> | <seq>“|”<regex>.
Will return <parse-tree> or (:ALTERNATION <parse-tree> <parse-tree>).
