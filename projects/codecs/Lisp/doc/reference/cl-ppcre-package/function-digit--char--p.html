Function Digit-Char-P

Part of:

package cl-ppcre
( digit-char-p < chr > )
Tests whether a character is a decimal digit, i.e. the same as
Perl's [\d]. Note that this function shadows the standard Common
Lisp function CL:DIGIT-CHAR-P.
